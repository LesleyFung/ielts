# 雅思阅读复盘 Skill（ielts-reading-review）

一个用于 **雅思阅读真题深度复盘** 的 Claude Skill。做完一篇剑桥真题后，把错题情况和原文/题目材料交给 Claude，它会以「资深雅思阅读名师」的身份，生成一份结构化的复盘报告——不只是告诉你对错，而是讲清楚**为什么错、原文在哪、同义替换是什么、下次怎么避坑**，并帮你沉淀一张可以长期积累的同义替换表和生词表。

---

## ✨ 它能做什么

- **错题深挖**：每道错题从四个维度拆解——错因诊断（点破出题思路）、定位分析（英文原句 + 位置）、同义替换（表格化）、长难句拆解（语法主干 + 地道翻译）。
- **对题快速过**：每道做对的题一行点出定位句和核心替换，帮你确认「是真会还是蒙对」。
- **文章结构梳理**：一句话概括每段大意 + 全文行文逻辑，训练篇章思维（尤其利于 Heading 题）。
- **题型避坑指南**：针对你本次错的题型，给出具体到方法的建议，而不是空话。
- **积累表沉淀**：错题+对题的同义替换汇总成一张总表，外加一份「四级以上重点词/熟词生义」生词清单，可直接复制进你的笔记。

## 🎯 触发方式

支持多种输入形式：**截图、照片、PDF、纯文字，甚至只口头描述错题**都行。

只要说类似这样的话，就会触发：

- 「帮我复盘这篇雅思阅读」
- 「剑14 Test1 Passage1 复盘一下」
- 「这几道题为什么错？」
- 「雅思阅读错题分析」

只需提供三类信息（缺哪个 Claude 会问哪个）：

1. **原文**（拍照/PDF/文字均可）
2. **错题情况**（题号、题型、正确答案、你的答案）
3. **题目内容**（截图里有就不用再给）

> ⚠️ **重要**：定位分析必须基于你**实际提供的原文**。请务必把原文一起上传，否则 Claude 只能做方向性的错因分析，无法给出精确的定位句和同义替换。

## 📋 复盘报告结构

```
① 本次成绩单        —— 正确率 + 错题分布 + 一句话判断
② 错题深挖          —— 每道错题：错因诊断 / 定位分析 / 同义替换 / 长难句拆解
③ 对题快速过        —— 确认是真会还是蒙对
④ 文章结构梳理      —— 每段大意 + 全文逻辑
⑤ 题型建议与避坑    —— 针对性方法
⑥ 同义替换积累表    —— 错题+对题全收录 + 生词清单
```

## 🚀 如何使用

1. 下载本仓库中的 `ielts-reading-review.skill` 文件。
2. 在 Claude 的 Skills 设置中导入该文件（或点击 skill 卡片上的 **Save skill**）。
3. 做完一篇雅思阅读后，把原文和错题情况发给 Claude，说一句「帮我复盘这篇阅读」即可。

## 📝 设计理念

复盘方法论贯穿整份报告的底层逻辑，绝大多数错题都能归到这四层之一：

1. 先看正确率，判断整体状态
2. 题干理解是否到位（词汇 + 句式）
3. 出题句（定位句）理解是否到位
4. 题干 ↔ 原文之间的同义替换是否识别出来

报告中的「错因诊断」会明确指出你错在了哪一层，让你下次遇到同类题能自己看穿套路。
